# **App Name**: FarsiFanConnect

## Core Features:

- Minimalist Landing Page: Create a dark-themed landing page with a button to access the registration/login form, with data being stored on Firebase and user auth by Firestore
- User Authentication: Implement user authentication via Firebase Authentication to securely register and log in users. Roles handled by Firestore.
- Role Assignment: Determine user role (creator/subscriber) based on Firestore data after signup.
- Form Submission: Collect full name, email, and password through the registration form and save on firestore
- Informational Landing Page Content: Write valuable information about the platform to display on the page.

## Style Guidelines:

- Primary color: Black (#000000) to evoke sophistication and exclusivity.
- Background color: White (#FFFFFF) to complement the black theme.
- Accent color: Light Blue (#ADD8E6) for interactive elements like buttons.
- Font: 'Alegreya', a serif font suitable for Farsi.
- Clean, single-column layout with clear call-to-action button.
- Subtle fade-in animation on page load and modal appearance.
